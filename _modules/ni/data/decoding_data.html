"""
.. module:: ni.data.decoding_data
   :platform: Unix
   :synopsis: Loads Data into a Panda Data Frame

.. moduleauthor:: Jacob Huth

Loads Data into a Panda Data Frame

Use like this:

	data = ni.data.decoding_data.DecodingData()
	plot(data.data.ix[(0,0,0):(0,3,-1)].transpose().cumsum())

.ix slices a portion of the data. The three indices are: Condition, Trial, then Cell. So the code above plots the cummulative firing rate for the first the trials of the first condition for all cells.


.. image:: _static/decoding_data_slice.png

or:

	data = ni.data.decoding_data.DecodingData()
	pylab.plot(data.data.xs(0,level='Condition').xs(0,level='Cell').ix[:5].transpose().cumsum())
	pylab.show()

With .xs one can slice at any level, ie. get specific trials and cells. this can be combined with .ix to get slices of data.
